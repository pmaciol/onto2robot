# onto2robot

The system for generating resoning engines based on ontologies


### Prerequisites
- Python 3.12+
- uv installed (`curl -LsSf https://astral.sh/uv/install.sh | sh`)

### Install Dependencies
```bash
uv sync --dev
```

### Run Tests
```bash
uv run pytest
```

### CLI Usage
After syncing, run the stub CLI:
```bash
uv run onto2robot --help
```
